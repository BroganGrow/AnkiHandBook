## 简介
Anki 是一款可以辅助人脑进行科学记忆的免费开源软件，它的本质就是一种 **闪卡**，正面是问题，背面是答案。闪卡有了 Anki 加持后，它就可以在相对合适的时间段里告诉你什么时候复习是最高效率的。
